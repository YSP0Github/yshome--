云凇学术 · 功能指南

本页概述平台提供的核心能力、操作流程与常见位置，方便新用户快速上手。 下方“部署与运行说明”仅对超级管理员可见。

1. 核心功能

  • 文献管理：上传 PDF / Office 文件，自动解析标题、作者、摘要与关键词，可自定义分类、标签、文献类型并支持批量导入导出。
  • 智能检索：多条件搜索、分类/年份筛选、排序与引用格式导出（GB/T、APA、MLA、BibTeX）。
  • 收藏与分享：收藏重要文献、生成公共分享页、导出引用格式；最近浏览与常用分类自动记录。
  • 用户体系：登录注册、头像上传、角色区分（普通用户/管理员/超级管理员），可一键切换主题与默认视图。
  • 后台管理：管理员可管理用户、上传记录、授权码，查看运行监控（在线人数、带宽、请求日志），配置安全策略与上传限制。

2. 使用流程

  1. 登录账号：使用管理员下发的账号或授权码登录；首次登录可在“个人资料”中完善昵称、头像与密码。
  2. 上传文献：点击“添加文献”，选择文件后检查解析结果( "上传文献" 接口的自动解析结果尚无法做到100%准确，建议手动检查并补全相关信息 )，补充分类、标签、文献类型并勾选版权声明即可提交。批量上传支持拖入多文件逐个编辑。
  3. 检索与阅读：在首页或搜索框输入关键词，可结合分类、年份、标签等条件筛选；点击文献标题进入详情页查看摘要、引用信息及附件。
  4. 收藏与共享：在文献详情页可收藏、导出引用格式、生成分享链接或下载附件。
  5. 管理员入口：拥有管理员权限的账号，可在右上角菜单进入“系统设置”“用户管理”“生成授权码”等后台页面。

3. 引用导出 & 第三方导入

所有用户都可以在文献卡片点击“引用”按钮，弹窗底部提供以下一键导出选项，适配常见文献管理软件：

BibTeX (.bib)

适用于 LaTeX、Zotero、Mendeley，在目标工具中选择“Import → BibTeX”。

EndNote (.enw)

EndNote 中执行 “File → Import → File”，格式选择 “EndNote Import”。

RefMan (.ris) / NoteExpress (.net) / NoteFirst (.txt)

都基于 RIS 结构，可在 NoteExpress、NoteFirst、RefMan 中选择 “导入 RIS” 直接导入。

Zotero (.json)

提供 Zotero JSON 包，执行 “文件 → 导入 → Zotero JSON” 即可批量写入。

导入失败通常与重复 DOI 有关，可在云凇学术中删除重复条目或改用其他格式重新导出。

4. 部署与运行说明（超级管理员可见）

常见位置

  • 上传文件存储：{{ upload_dir }}
  • 应用日志：{{ log_path }}

环境变量

  • YSXS_URL_PREFIX=/YSXS（部署在子路径时必填）
  • YSXS_DATABASE_URI=sqlite:////home/YSP/var/ysxs/ysxs.db 或其他数据库 URI
  • YSXS_UPLOAD_DIR=/home/YSP/var/ysxs/uploads（上传目录需可写）

systemd 服务示例

[Service]
User=YSP
WorkingDirectory=/home/YSP/yshome
Environment="YSXS_DATABASE_URI=sqlite:////home/YSP/var/ysxs/ysxs.db"
Environment="YSXS_UPLOAD_DIR=/home/YSP/var/ysxs/uploads"
ExecStart=/home/YSP/miniconda3/envs/YSZJ/bin/gunicorn -w 2 -b 127.0.0.1:5050 YSXS.app:app
Restart=always

修改后执行 sudo systemctl daemon-reload && sudo systemctl restart ysxs-gunicorn。

Nginx 关键配置

  • proxy_pass http://127.0.0.1:5050/YSXS/; 并设置 X-Forwarded-Prefix=/YSXS。
  • client_max_body_size 50m;（允许大文件解析与上传）。
  • 静态资源 /YSXS/static/ 建议使用 alias /home/YSP/yshome/YSXS/static/。

数据库初始化

export FLASK_APP=YSXS.app
flask db upgrade
