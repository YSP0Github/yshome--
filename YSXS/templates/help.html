快速入门

如何注册与登录？

  1. 点击页面右上角的"登录/注册"按钮
  2. 在登录页面点击"立即注册"链接
  3. 填写用户名、邮箱和密码，完成注册
  4. 使用注册的账号密码登录系统

系统主要功能有哪些？

  • 文献上传与管理：支持多种格式的学术文献上传
  • 文献分类：可自定义分类管理文献
  • 文献检索：通过标题、作者、关键词等快速查找
  • 收藏功能：标记重要文献以便快速访问
  • 个人资料管理：修改个人信息和系统设置

文献管理

如何上传文献？

  1. 点击页面右上角的"上传文献"按钮
  2. 在弹出的上传窗口中，拖放文件或点击"浏览文件"选择本地文献
  3. 填写文献信息：标题、作者、期刊/会议、年份等
  4. 选择文献分类和类型
  5. 填写关键词和摘要（可选，但建议填写以便检索）
  6. 点击"确认上传"按钮完成上传

支持格式：PDF, DOC, DOCX, PPT, PPTX, TXT（最大50MB）

如何搜索文献？

系统提供多种搜索方式：

  • 在顶部搜索框输入关键词，支持标题、作者、关键词搜索
  • 通过左侧分类栏筛选特定分类的文献
  • 使用"我的收藏"查看已标记的文献
  • 通过"最近浏览"快速访问近期查看过的文献

如何管理文献分类？

您可以通过以下方式管理文献分类：

  • 上传文献时选择现有分类或创建新分类
  • 在"文献分类"页面查看所有分类及包含的文献数量
  • 点击分类名称可查看该分类下的所有文献

如何导出为 EndNote / NoteExpress 等软件使用？

  1. 在任何文献卡片中点击引用按钮打开“文献引用格式”弹窗。
  2. 在弹窗底部的“导出到常用软件”区域选择需要的格式并下载。
  3. 在目标软件中执行“导入/Import”并选择对应的文件类型。

BibTeX (.bib)

适用于 LaTeX、Zotero 等，Zotero 中选择 “文件 → 导入 → BibTeX”。

EndNote (.enw)

在 EndNote 中执行 “File → Import → File”，格式选择 “EndNote Import”。

RefMan (.ris) / NoteExpress (.net) / NoteFirst (.txt)

以上均为 RIS 结构，可在 NoteExpress、NoteFirst、RefMan 等软件中通过 “Import RIS” 或 “导入参考文献” 直接导入。

Zotero (.json)

提供 Zotero JSON 包，Zotero 中选择 “文件 → 导入 → Zotero JSON”。

如果用户已导入相同 DOI 的文献，可先在“批量引用”区域批量去除重复再导出。

账户设置

如何修改个人资料？

  1. 点击右上角头像，在下拉菜单中选择"个人资料"
  2. 在个人资料页面可以修改用户名和邮箱
  3. 点击头像下方的相机图标可以更换头像
  4. 修改完成后点击"保存基本信息"按钮

如何修改密码？

  1. 在个人资料页面找到"修改密码"区域
  2. 输入当前密码和新密码
  3. 点击"修改密码"按钮完成修改

如何设置系统通知？

  1. 点击右上角头像，在下拉菜单中选择"系统设置"
  2. 在左侧导航中选择"通知设置"
  3. 勾选需要接收的通知类型
  4. 设置会自动保存

如何注销账户？

  1. 在系统设置页面找到"账户设置"区域
  2. 勾选"我已了解删除账户将清除所有数据且无法恢复"
  3. 点击"注销账户"按钮
  4. 在确认弹窗中点击"确认"完成注销

注意：账户注销后所有数据将被永久删除，且无法恢复，请谨慎操作
