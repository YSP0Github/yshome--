# PendulumLab

Standalone browser demo for double, triple, and quadruple pendulums.

## Run

```powershell
cd "G:\AI World\PendulumLab"
python run.py
```

Open:

```text
http://127.0.0.1:8787
```

## Features

- Double, triple, and quadruple pendulum modes
- Pause/resume, reset, random pose, clear trail
- Adjustable simulation speed, gravity, damping, and trail length
- Drag any bob to change the initial pose
- Pure browser canvas animation served by a tiny Python HTTP server
