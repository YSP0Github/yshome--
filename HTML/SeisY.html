软件概述

SeisY 是一个综合性的地震数据处理和分析平台，基于 PyQt6 构建，当前提供 Windows 桌面版，适用于地球物理研究与教学。

关于 SeisY

SeisY 面向地震、月震与行星地震数据的处理、分析与可视化，支持 MiniSEED、SAC、SEG-Y、SEG2、ASCII、TXT、CSV 等 8 种数据格式的导入导出，提供从数据预处理到高级分析的完整工作流。

软件集成波形处理、频谱分析、震源定位、射线追踪、简正模分析、月震工作流和 Apollo 数据管理等 15+ 专业工具模块，支持地球、月球和自定义行星速度模型。

核心功能

SeisY 提供从数据预处理到高级分析的完整功能链

波形数据处理

20 余种处理操作，覆盖从原始数据到分析就绪的完整预处理链。

  • 去均值 / 去趋势 / 去仪器响应
  • 10 种插值方法 / 6 种加窗函数
  • 带通 / 中值滤波 / 归一化

频谱分析

完整的频谱分析工具链，从振幅谱到 PSD 到时频联合分析。

  • 振幅谱 / 功率谱密度 (PSD)
  • Welch 平均 / 时频图
  • 滑动相关分析 / SNR 评估

震源定位

完整的震源反演定位窗口，支持多种行星速度模型。

  • 台站、波形、拾取、模型一体化
  • 地球 / 月球 / 自定义行星模型
  • 加权最小二乘反演与结果可视化

射线追踪

分层速度模型的射线路径分析与走时计算工作台。

  • 正演模式与反求模式
  • 起射角搜索与传播几何分析
  • 分层模型路径积分可视化

简正模分析

从模型到观测的完整自由振荡与简正模分析链。

  • 模态枚举、频率估计、Q 与阻尼
  • 特征函数、震源激发与台站合成
  • 观测谱峰检测与理论匹配

月震工作流

面向 Apollo 月震数据的多步骤反演流程与数据管理系统。

  • 月震反演：下载→拾取→定位→分析
  • Apollo 批量下载与数据管理
  • PDDM 去噪 / 噪声提取 / ULF 重构

专业工具

SeisY 内置的专业工具模块截图展示

震源定位器

台站、波形、拾取、速度模型、初始震源猜测、结果与地图联动的完整定位流程。

行星台站地图

台站/震源管理、震中距计算、到时预测、CSV 与图像导出，可联动射线追踪与简正模。

射线追踪工作台

射线路径、起射角搜索与传播几何分析，支持分层速度模型的正演与反求。

简正模工作台

模型、模态、激发与合成分析，提供精确球贝塞尔求根与观测谱对照。

模型设计器

1D / 准 1D 分层模型设计、模型体检、导出及发送到简正模工作台。

Apollo 月震系统

Apollo 月震数据的批量下载、同步、提取、导出和覆盖查看，支持 XA 网络 S12/S14/S15/S16。

技术栈与使用指南

技术架构

PyQt6 桌面框架

基于 PyQt6 的桌面 GUI，当前提供 Windows 版本。

ObsPy 数据引擎

地震学标准数据处理库，支持 MiniSEED、SAC 等格式读写。

pyqtgraph 实时绘图

高性能波形渲染，适合 10-60 FPS 的实时滚动显示。

OpenGL 天体可视化

地球、月球、火星等球体渲染，联动台站与震源显示。

走时计算工具

自主开发的射线理论走时计算工具，并与 TauP 对比验证，可用于震源定位与相位预测。

快速上手

常规数据处理流程

  1. 1使用"导入文件"导入波形数据
  2. 2在事件列表中选中目标事件
  3. 3依次执行去均值、去趋势、滤波和去除仪器响应
  4. 4使用"绘制当前数据流"查看波形
  5. 5使用"绘制振幅谱"查看频谱
  6. 6使用"保存数据"或"保存项目"保存结果
