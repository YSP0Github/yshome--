# SeisY v0.1.6 — 重大更新版本

## ✨ 版本亮点

SeisY v0.1.6 是一次重要更新。  
相较于此前的 **v0.1.3** 主要以 Python / wheel 分发为主、安装门槛较高的方式，这一版本首次提供了更适合普通用户直接使用的 **Windows 安装包** 和 **便携版可执行文件**。

这意味着：

- 不再必须先手动配置 Python 环境
- 不再依赖命令行安装 `whl`
- 普通用户可以更方便地直接启动 SeisY
- 更适合教学演示、实验室电脑部署与日常分发使用

---

## 🚀 本次重大更新

### 1. 首次提供 Windows 安装版
现在可以直接通过安装程序完成部署，降低了初次使用门槛，更适合第一次接触 SeisY 的用户。

### 2. 首次提供 Windows 便携版
新增便携版压缩包，解压后即可运行 `SeisY.exe`。  
适合以下场景：

- U 盘携带
- 实验室电脑临时运行
- 无需安装的快速演示
- 多台设备快速分发

### 3. 发布方式更接近正式桌面软件
从过去偏开发者导向的 Python 包分发，升级为更面向最终用户的桌面软件分发方式。这是 SeisY 在可用性和交付方式上的一次明显进步。

---

## 📦 下载说明

本版本提供以下两个 Windows 发行文件：

### `SeisY-0.1.6-Setup.exe`
**推荐普通用户使用**

适合：

- 首次安装
- 希望获得标准安装流程的用户
- 需要更省心部署方式的用户

### `SeisY-0.1.6-Portable.zip`
**适合无需安装的使用场景**

解压后运行其中的 `SeisY.exe` 即可。  
请保留解压后的完整目录结构，不要只单独移动 exe 文件。

适合：

- 实验室电脑
- U 盘便携使用
- 临时演示
- 无管理员权限环境

---

## 🔧 与旧版本相比

相较于 **v0.1.3**，本版本最大的变化不是单一功能点，而是**软件分发与使用体验的整体升级**：

- 从 `whl` 安装转向更友好的桌面软件发布方式
- 从“需要配置 Python 环境”转向“下载即可运行 / 安装”
- 更适合非开发者、同学、老师和合作使用者直接体验

这也是 SeisY 从“开发中的工具集合”向“更完整、更易用的软件产品”迈出的重要一步。

---

## 🧪 使用建议

- **第一次使用**：建议下载安装版 `Setup.exe`
- **临时运行 / 拷贝使用**：建议使用便携版 `Portable.zip`
- 若系统弹出安全提示，可确认来源后继续运行
- 便携版请勿只单独复制 `SeisY.exe`，应保留整个解压目录
