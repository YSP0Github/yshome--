# SeisY v0.1.6 正式发布

SeisY v0.1.6 现已正式发布。  
这是一次具有阶段意义的重要更新版本。

相较于此前版本主要以 Python 包形式分发、需要用户自行配置运行环境的方式，v0.1.6 首次提供了面向 Windows 用户的**安装版程序**与**便携版可执行文件**。这标志着 SeisY 的发布方式从偏开发与内部使用，进一步转向更完整、更易用的桌面软件交付形态。

本次更新的重点，不仅在于功能层面的持续完善，更在于软件可获取性、可部署性与实际使用体验的明显提升。

---

## 一、本版本的主要意义

SeisY v0.1.6 是一个重要的发布节点。

在此前版本中，用户通常需要先准备 Python 环境、安装依赖，并通过 `whl` 包或命令行方式启动程序。这种方式虽然适合开发与测试，但对普通使用者、教学演示场景以及跨设备部署而言，仍然存在较高门槛。

从 v0.1.6 开始，SeisY 首次提供：

- **Windows 安装版（Setup）**
- **Windows 便携版（Portable）**

这意味着用户现在可以通过更直观、更接近常规桌面软件的方式获取和使用 SeisY，而不必再将环境配置作为使用前提。

---

## 二、本次发布的核心更新

### 1. 首次提供 Windows 安装版
本版本新增标准 Windows 安装包，用户可通过安装程序完成部署。  
这一更新显著降低了首次使用门槛，也使 SeisY 更适合用于课程展示、实验室电脑部署及向非开发用户分发。

### 2. 首次提供 Windows 便携版
本版本同时新增便携版压缩包。用户在解压后即可直接运行 `SeisY.exe`。  
这一形式更适合：

- 无需安装的快速试用
- U 盘携带与移动分发
- 实验室或机房环境中的临时运行
- 无管理员权限条件下的软件使用

### 3. 软件交付方式显著升级
与旧版本相比，v0.1.6 在软件发布层面实现了明显进步。  
SeisY 不再仅仅依赖开发者友好的 Python 包分发方式，而是开始向更标准的桌面软件发布模式过渡。这对于提升软件的可访问性、推广性与实际落地能力具有重要意义。

---

## 三、发行文件说明

本次发布包含以下 Windows 版本文件：

### `SeisY-0.1.6-Setup.exe`
**适合大多数用户，推荐优先使用。**

适用场景：

- 首次安装
- 日常桌面使用
- 希望通过标准安装流程完成部署的用户

### `SeisY-0.1.6-Portable.zip`
**适合无需安装、希望快速运行的用户。**

使用方式：

- 下载压缩包并解压
- 在解压目录中运行 `SeisY.exe`

注意事项：

- 请保留解压后的完整目录结构
- 不建议只单独复制 `SeisY.exe`，否则可能因缺少运行依赖而无法正常启动

---

## 四、与旧版本相比的变化

相较于早期版本（如 v0.1.3），本版本最重要的变化并不只是局部功能调整，而是整体使用方式的升级：

- 从依赖 Python 环境配置，转向更直接的软件安装与运行方式
- 从面向开发与测试，转向更适合普通用户获取与使用
- 从“项目型工具”进一步迈向“可交付软件”形态

因此，v0.1.6 可以视为 SeisY 在产品化与可用性方向上的一次关键推进。

---

## 五、适用建议

- **普通用户 / 首次使用者**：建议下载并使用 `Setup.exe`
- **临时运行 / 移动分发 / 无安装权限环境**：建议使用 `Portable.zip`
- 如系统弹出安全提示，请在确认来源可信后继续运行
- 便携版请务必保留完整解压目录
